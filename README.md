# 駕訓班筆試練習 PWA v5

這版在既有題庫基礎上新增：

- 題目積分：每題預設 0 分，答對 +1、答錯 / 逾時 / 不會 -1
- 積分篩選：可依 `>`、`<`、`=` 與自填門檻抽題
- 每題限時：逾時自動判錯
- 作答後自動跳題：秒數可調，0 表示不自動
- 「不會」按鈕：直接判錯並進下一題
- 積分 > 1 覆蓋率統計
- 單字卡複習模式
- 學習獎勵與徽章

## 啟動方式
site is live at https://haxcant.github.io/driver-quiz-mobile-202604/
### Windows
直接雙擊：

`start_server.bat`

### 或手動

```bash
python -m http.server 8080
```

然後開啟：

`http://127.0.0.1:8080`

## 建議使用方式

1. 先選 **正式筆試模式（普通小型車）**
2. 用一般練習或模擬考建立初始積分
3. 再用 **積分篩選 `< 1`** 專打弱題
4. 最後切到 **單字卡複習** 反覆掃過低分題

## 注意

如果你以前安裝過舊版 PWA：

- 建議先刪掉舊主畫面捷徑，或
- 至少在瀏覽器用 `Ctrl + F5` 強制重整

這版 service worker 已更新版本號，但若舊快取未清理，仍可能暫時看到舊畫面。


## 新增：駕駛人手冊解答說明

- 作答後會顯示《駕駛人手冊》對應章節的說明與頁碼。
- 標誌題以手冊附錄與標誌章節做說明。
- 法規題以手冊中最相近的規則／安全駕駛段落做說明。
- 機械常識題若手冊無直接條文，會明確提示其屬手冊主體外的延伸題。


## 本版修正
- 是非題選項統一改為與答案同語彙（例如：正確 / 錯誤）。
- `依圖示判斷` 題型在是非題中也會顯示題圖。
- 新增作答中暫停 / 繼續按鈕。
- 自動跳題拆成「答對後」與「答錯後」兩組獨立秒數。
- 學習獎勵區塊移到答題區塊下方。


## 完整學習記憶匯出 / 匯入

- 可匯出完整記憶 JSON，包含每題積分、答題次數、錯題本狀態、累計統計、目前設定，若有進行中的題組也會一併保存。
- 匯入時可選擇「覆蓋目前本機記憶」或「與目前記憶合併」。
- 建議定期備份此 JSON，以便跨裝置或重灌後恢復進度。
