# 台灣小型汽車考照筆試練習 v20.2

這是一個以 **盡量按少按鈕、可直接進入作答、輕鬆整理題目、隨時用手機/電腦複習** 為目標設計的汽車駕照筆試練習網站。

## 第一次使用
打開網址 https://haxcant.github.io/driver-quiz-mobile-202604/
第一次使用者只須點擊 **開始練習** 即可開始作答。  
系統已提供預設題數、預設轉場與基本流程，不必先調整進階設定。

## 主要功能
- 普通小型車正式筆試題庫
- 正式筆試 + 機械常識加強模式
- 標明題目出處
- 完整延伸模式
- 一般練習、只練錯題、模擬考、單字卡
- 看圖選名稱、看名稱選圖、混合題型
- 錯題本、積分追蹤、正分覆蓋率、學習徽章
- 《駕駛人手冊》對照說明與搜尋此題
- JSON 匯出／匯入完整記憶與錯題資料
- Firebase 手動雲端同步（白名單）
- 同步前本機備份與可還原
- 可分別設定答對、答錯後自動跳題時間
- 提示音量調整（0%～500%）與音量測試
- 快捷積分篩選按鈕（考沒考過題／考錯過題目）
- 手機圖片載入失敗自動重試與快取修補

## 平台相容
- 電腦瀏覽器
- 平板瀏覽器
- 手機瀏覽器
- 支援 PWA 安裝到主畫面（視瀏覽器支援而定）

## 操作設計重點
- 預設可直接開始作答
- 進階設定可摺疊，避免畫面過於複雜
- 所有重要說明至少保留在首頁版本資訊或 README 的其中一處，不因版面精簡而整體消失
- 雲端同步預設收合，個資預設隱匿
- 快捷篩選按鈕會直接套用進階設定中的積分篩選
- 雲端同步採手動上傳／下載，避免過度占用第三方資源額度
- 僅需網址即可進入系統複習，免安裝、高相容、上手快

## 積分規則
- 答對：+1 分
- 答錯 / 逾時 / 不會：-1 分
- 正分覆蓋率可幫助你判斷整體熟練度

## 預設答題節奏
- 每題限時預設：15 秒
- 答對後自動跳題預設：1 秒
- 答錯後自動跳題預設：4 秒

## 雲端同步說明
- 未登入時仍可正常練題與使用 JSON 匯入匯出
- 登入後只會顯示雲端備份摘要，不會直接覆蓋本機
- 只有按下 **下載雲端備份** 時，才會詢問是否載入
- 若要啟用雲端同步，需將 **名稱 / 信箱 / UID** 提供給擁有者加入白名單

## 備份提醒
- 建議使用者 **定期下載 JSON 備份檔**
- 雲端同步屬於第三方託管資源，可能受額度、網路或服務狀況影響
- 即使已啟用雲端同步，仍建議保留本機 JSON 備份，以降低資料遺失風險

## 建議使用方式
1. **先直接按開始練習，優先提升覆蓋率**
   - 先讓更多題目至少看過一次，不要一開始就只刷錯題
2. **覆蓋率提升後，再加強錯題與低分題**
   - 可直接使用快捷按鈕：
     - **考沒考過題（積分 = 0）**
     - **考錯過題目（積分 < 0）**
   - 也可在進階設定中自行調整積分篩選
3. **最後用模擬考驗證整體熟悉度**
4. 需要跨裝置時，再手動上傳雲端備份或匯出 JSON


## 圖片載入提醒
- 若手機偶發某題圖片未顯示，系統會先自動重試一次
- 若仍失敗，建議重新整理頁面或先匯出 JSON 備份後再重新開啟


## 完整記憶匯入模式
- 覆蓋：以匯入檔完整取代本機資料。
- 覆蓋率優先合併（放水取高）：同一題兩邊不同時，優先保留較高覆蓋率／較高分的一側，適合快速刷題。
- 保守合併（低分優先）：同一題兩邊不同時，優先保留較低分、較需要複習的一側，適合避免弱點被沖淡。
- 三種模式都不再直接相加統計，避免同源資料互相合併後數值異常膨脹。


## JSON 匯入安全與大小限制
- 匯入前會先自動保存一份「匯入前本機備份」。
- 系統只接受白名單欄位，未知欄位會忽略；若檔案包含 `__proto__`、`prototype`、`constructor` 等危險鍵，會直接拒絕匯入。
- 匯入後總統計會重新計算，不直接信任舊檔內附的總表。
- 本機匯入大小上限：
  - 一般 JSON：8 MiB
  - 錯題列印版 HTML：16 MiB
- 雲端備份會先自動精簡：
  - 只保留有實際作答／有變化的題目
  - 移除可重算總表與不必要欄位
  - 分塊後上傳（最多 10 塊，總量約 2.40 MiB）
- 若備份仍超限，建議改用本機 JSON 匯出保存。

## 舊 JSON 修整與平坦化
- 匯入舊版完整記憶 JSON 時，系統會自動重算總統計，修整不一致的 meta。
- 若舊檔因多次合併導致分數累積過大，可使用「平坦化積分（±1）」功能：
  - 分數大於 1 的題目改為 1
  - 分數小於 -1 的題目改為 -1
- 此功能需雙重確認，不會改變作答次數，只會修整題目積分。
