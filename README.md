# 台灣小型汽車考照筆試練習 v18.1

這是一個以 **盡量按少按鈕、可直接進入作答、輕鬆整理大量題目、利用手邊設備隨時複習** 為目標設計的汽車駕照筆試練習網站。

## 第一次使用
第一次使用者只須點擊 **開始練習** 即可開始作答。其餘像是題數、分類、積分篩選、雲端同步都可以之後再調整。

## 主要功能
- 普通小型車正式筆試題庫
- 正式筆試 + 機械常識加強模式
- 完整延伸模式
- 一般練習、只練錯題、模擬考、單字卡
- 看圖選名稱、看名稱選圖、混合題型
- 錯題本、積分追蹤、正分覆蓋率、學習徽章
- 《駕駛人手冊》對照說明與搜尋此題
- JSON 匯出／匯入完整記憶與錯題資料
- Firebase 手動雲端同步（白名單）
- 同步前本機備份與可還原
- 針對答題正確、錯誤設置不同轉場時間
  
## 平台相容
- 電腦瀏覽器
- 平板瀏覽器
- 手機瀏覽器
- 支援 PWA 安裝到主畫面（視瀏覽器支援而定）

## 操作設計重點
- 預設可直接開始作答
- 進階設定可摺疊，避免畫面過於複雜
- 雲端同步預設收合，個資預設隱匿
- 快捷篩選按鈕可快速套用 `積分 = 0` 與 `積分 < 0`
- 雲端同步採手動上傳／下載，避免過度占用第三方資源額度
- 雲端採用google登入，免記密碼、增加安全性

## 積分規則
- 答對：+1 分
- 答錯 / 逾時 / 不會：-1 分

## 雲端同步說明
- 未登入時仍可正常練題與使用 JSON 匯入匯出
- 登入後只會顯示雲端備份摘要，不會直接覆蓋本機
- 只有按下 **下載雲端備份** 時，才會詢問是否載入
- 若要啟用雲端同步，需將名稱 / 信箱 / UID 提供給擁有者加入白名單

## 建議使用方式
1. 先直接按 **開始練習** 熟悉流程
2. 再用快捷篩選或進階積分篩選找弱點
3. 用模擬考驗證整體熟悉度
4. 需要跨裝置時再手動上傳雲端備份或匯出 JSON
